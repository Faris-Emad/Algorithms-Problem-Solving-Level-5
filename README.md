# Algorithms & Problem Solving Level 5 - Data Structures

هذا المستودع يحتوي على المشاريع العملية والتطبيقات الخاصة بـ **الكورس 13 (مستوى حل المشكلات 5 - هياكل البيانات)** من سلسلة قنوات **Programming Advices** للمهندس **محمد أبو هدهود**.

تركز هذه المشاريع على بناء هياكل البيانات الأساسية (Data Structures) من الصفر باستخدام لغة **C++** لتعميق الفهم البرمجي لكيفية عمل هذه الهياكل خلف الكواليس.

## 📂 محتويات المستودع (المشاريع)

### 1. Project 1: Double Linked List Class
بناء صنف (Class) متكامل يمثل القائمة المزدوجة الترابط (Doubly Linked List)، ويحتوي على كافة العمليات الأساسية مثل الإضافة، الحذف، التحديث، والبحث (Insert, Delete, Update, Find).

### 2. Project 2: MyQueue
تنفيذ هيكل بيانات الطابور (Queue) اعتماداً على الـ Doubly Linked List الذي تم بناؤه في المشروع الأول، مع تطبيق مبدأ (FIFO - First In First Out).

### 3. Project 3: MyStack Requirements
تنفيذ هيكل بيانات المكدس (Stack) بالاعتماد على الـ MyQueue وبناء عملياته الأساسية (Push, Pop, Top, Bottom) وتطبيق مبدأ (LIFO - Last In First Out).

### 4. Project 4: MyDynamicArray
بناء صنف للمصفوفات الديناميكية (Dynamic Array) من الصفر، لتجاوز قيود المصفوفات العادية ذات الحجم الثابت في C++، متضمناً دوال مثل الإضافة في النهاية، وتغيير الحجم، والحذف، والإدراج (Resize, InsertAt, DeleteAt).

### 5. Project 5: MyQueueArr
إعادة بناء هيكل الطابور (Queue) ولكن هذه المرة بالاعتماد على المصفوفة الديناميكية (Dynamic Array) بدلاً من القوائم المترابطة، مع مراعاة الأداء وتنفيذ جميع العمليات الأساسية.

### 6. Project 6: MyStackArr
إعادة بناء المكدس (Stack) بالاعتماد ع��ى المصفوفة الديناميكية (Dynamic Array)، لتخزين العناصر واسترجاعها بطريقة فعالة.

### 7. Project 7: MyString
بناء صنف خاص بالنصوص (String Class) يحتوي على دوال وعمليات متقدمة للتعامل مع السلاسل النصية (مثل التقطيع Split، الدمج Join، استبدال الكلمات، وغيرها من الخصائص المتقدمة).

### 8. Project 8: (Undo/Redo & Advanced Operations)
تطبيق عملي متقدم (غالباً نظام التراجع والإعادة Undo/Redo) يدمج ما تم تعلمه في هياكل البيانات (Stacks & Queues) لعمل محاكاة لأنظمة التراجع في البرامج الحقيقية.
